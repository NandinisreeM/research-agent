AI-Powered Account Research Agent


A full-stack AI application that automates company research. Built with Node.js, React, Tailwind CSS, and Google Gemini AI.

Features
- AI-Driven Insights: Generates product summaries, GTM strategies, and competitor lists.
- Smart Formatting: Uses Gemini 1.5 Flash to return structured JSON data.
- Modern UI: Professional dashboard with glassmorphism and skeleton loading states.
- Fast Performance: Built with Vite for nearly instant frontend updates.

Tech Stack
- Frontend: React, Tailwind CSS, Lucide Icons, Vite.
- Backend: Node.js, Express.js, CORS, Dotenv.
- AI Model: Google Gemini 1.5 Flash (via Google Generative AI SDK).
